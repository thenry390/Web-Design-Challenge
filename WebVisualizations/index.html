    • Summary: Latitude vs. X

    The purpose of this project was to analyze how weather changes as you approach the equator. To accomplish this analysis, we first pulled data from the OpenWEatherMap API to assemble a dataset on over 500 cities. Afer assembling the dataset, we used Matplotlib to plot various aspects o fthe weather versus latitude. Facters considered included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correllations witnessed.
